Projeto prático de algoritmo e estruturas de dados.

O projeto foi pensado e criado usando a linguagem c
para sistemas windowns e linux
usando a ide devc++ e VScode

Pasta main:
    Contém o arquivo:
        main.c apenas chamando a função do jogo.

Pasta componentes:
    Contém os arquivos:
        carta.c que contém o TAD representando as cartas.
        baralho.c que será responsável por criar os baralhos.
        embaralhar.c que sera responsável por embaralhar os 2 baralhos.
        jogo.c que contém o loop principal do jogo, adição de pontos e verificação de vitoria.
        incluir_carta.c que ira inserir a carta nos espaços e verificar se o jogador formou pares.
   
Pasta estruturas:
    Contém os arquivos:
        deque.c contém todo o código para o funcionamento da estrutura deque.
        fila.c contém todo o código para o funcionamento da estrutura fila.
        pilha.c contém todo o código para o funcionamento da estrutura pilha.
